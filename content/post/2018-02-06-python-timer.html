栾佳硕给出的作业，要求三秒钟打印出一个随机小数:

# -*- coding: utf-8 -*-
import random
import time
seconds = 3
counter = 1
while counter <= 5:
    print("第%i个两位小数是%.2f" %(counter,random.uniform(0,1)))
    counter += 1
    time.sleep(seconds)
## 第1个两位小数是0.83
## 第2个两位小数是0.31
## 第3个两位小数是0.18
## 第4个两位小数是0.77
## 第5个两位小数是0.64
